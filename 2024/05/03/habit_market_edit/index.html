仓位说明：
兼容最小步：账户余额较少时，仓位使用最小值，步长减少并且均匀散开。
小重大轻：小账户仓位更重。

运行这个脚本，即可显示仓位大小说明。
Scripts\_OrderTest\OptimizeAccountLot.mq5

仓位修改：

如上图，修改 MM=0.06。即是每次批量下单总仓位为 6%。
然后重新编译。

止赢止损：
如上图，修改
STOPLOSS=150; // 止损150点，实测是$1.5
TAKEPROFIT=150; // 止赢300点

批量开单范围和间隔：
如上图，修改
RANGE=200; // 步长范围200点，在这个价格区间上下批量下单，
RTOTAL=20; // 步长40点，即10单

修改第一单价格偏移：
如上图，修改
LEVEL=40; // 与当前价格相差 $0.4, 开始开单。

重新编译

步骤：
a. 打开IDE，修改脚本参数
b. 右键 _Orderxx 文件夹，右键编译
c. 打开MT5软件，右键脚本刷新，需要时点击脚本运行
