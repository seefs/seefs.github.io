Modify parameters

position description:
Compatible minimum step: When the account balance is small, the position uses the minimum value, the step size is reduced and evenly spread.
Small and light: Small accounts have heavier positions.

Run this script and the position size description will be displayed.
Scripts\_OrderTest\OptimizeAccountLot.mq5

position modification:

As shown in the figure above, modify MM=0.06. That is, the total position of each batch order is 6%.
Then recompile.

Take Profit and Stop Loss:
As shown in the figure above, modify
STOPLOSS=150; //The stop loss is 150 pips, and the measured is $1.5
TAKEPROFIT=150; //Take profit of 300 points

range and interval of bulk billing:
As shown in the figure above, modify
RANGE=200; //The step size range is 200 points, and the order is placed in batches in this price range.
RTOTAL=20; //The step size is 40 points, that is, 10 orders

modify the price offset of the first order:
As shown in the figure above, modify
LEVEL=40; //A difference of $0.4 from the current price opens a ticket.

Recompilation

steps:
a. Open the IDE and modify the script parameters
b. Right-click on the _Orderxx folder and right-click on compilation
c. Open the MT5 software, right-click on the script to refresh, and click the script to run if needed
